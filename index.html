Página web

Es un documento que se ve en el navegador. Puede tener texto, imágenes, videos, botones y enlaces.

Sitio web

Es el conjunto de varias páginas relacionadas que comparten un mismo dominio y un mismo propósito.

Idea clave

Un sitio web es como una biblioteca. Cada página web es uno de los libros que vive dentro de ella.

Tema 1

Página web y sitio web

Haz clic para comparar. La idea es separar lo individual de lo colectivo.

Página web

Una página web es un documento digital que se abre en un navegador. Tiene una dirección propia y suele mostrar información específica, por ejemplo: "Inicio", "Contacto" o "Galería".

Comparación sencilla: una página web es como una hoja de un cuaderno. Tiene contenido puntual y cumple una función concreta.
1
Unidad individual. Se ve como una sola pantalla o documento.
2
Tema específico. Puede hablar solo de una sección del negocio o proyecto.
3
Ejemplo. La página "Contacto" dentro de una escuela en línea.

Tema 2

Conceptos básicos del diseño web

Explora cada concepto y luego usa el laboratorio de URL para reconocer sus partes.

Dominio

El dominio es el nombre fácil de recordar de un sitio web. En lugar de memorizar números del servidor, usamos un nombre como miescuela.com.

Función: identificar
Ejemplo: google.com
Idea clave: nombre del sitio
Analogía: el dominio es como el nombre del lugar al que quieres llegar.

Tema 4

Estándares de diseño web (W3C)

El W3C propone recomendaciones para que la web sea más clara, compatible, accesible y ordenada.

¿Qué es el W3C?

Es el World Wide Web Consortium, una organización que promueve estándares para HTML, CSS, accesibilidad y más.

¿Por qué importa?

Porque ayuda a que una página funcione mejor en distintos navegadores, dispositivos y contextos de uso.

¿Qué recomienda?

Usar código limpio, etiquetas semánticas, diseño adaptable, accesibilidad y separación entre estructura y estilo.
